# 六爻神卦 · 铜钱占卜

一个纯前端的六爻铜钱占卜网站：输入所求问之事，以**三枚铜钱摇六次**起卦，生成本卦与变卦，并可**一键复制完整卦象交给任意 AI 解读**。界面偏神秘学风格，含星空、八卦环、旋转太极与铜钱 3D 翻转动画。

## 特性

- 🪙 **铜钱起卦**：3 枚铜钱、摇 6 次，带 3D 翻转弹跳动画与合成音效（可开关，无外部音频文件）
- ☯ **完整卦象**：自动推算本卦、变卦、动爻，覆盖全部 64 卦（已校验卦序 1–64 无重无漏）
- 📋 **一键复制**：把「问题 + 起卦时间 + 起卦方式 + 本卦/变卦 + 六爻明细 + 动爻 + 解读指令」整理成纯文本，粘贴给 ChatGPT / Claude / 文心一言等即可获得解读
- ✨ **纯静态**：原生 HTML / CSS / JS，无任何依赖

## 起卦规则

以铜钱摇卦，约定 **背（花面/无字）为阳记 3，字（有字面）为阴记 2**：

| 三枚结果 | 爻值 | 含义 |
|---|---|---|
| 三背 | 9 | 老阳（动 → 变阴） |
| 两背一字 | 8 | 少阴（静） |
| 一背两字 | 7 | 少阳（静） |
| 三字 | 6 | 老阴（动 → 变阳） |

每枚铜钱正反各 50%（`Math.random()`），三枚独立，符合真实铜钱的概率分布。自下而上摇 6 次成卦。

直接用浏览器打开 `index.html` 即可（`file://` 亦可），无需服务器。
